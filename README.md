## Scaffolding project

- Get repo `git clone https://github.com/StephenGrider/ReduxSimpleStarter.git`
- Change to folder `ReduxSimpleStarter`
- Install dependancies `npm install`
- Run project `npm start`
- Visit `localhost:8080`
